# 경찰서 또는 면허 담당 부서 전산 반영 요청문

아래 문안은 행정심판 인용 재결 후 결격 해제 또는 처분 취소 내용이 전산에 반영되도록 요청하는 예시입니다.

```text
안녕하세요.

행정심판 인용 재결을 받아 운전면허 결격 해제 전산 반영을 요청드립니다.

첨부한 재결서를 확인하시어 운전면허 결격 정보가 정정 또는 반영될 수 있도록 처리 부탁드립니다.

사건 관련 개인정보는 별도 첨부자료에 기재하였으며, 필요하신 추가 자료가 있으면 안내 부탁드립니다.

감사합니다.
```

## 주의

* 공개 저장소에는 실제 재결서 PDF를 올리지 마십시오.
* 실제 제출 시에는 이름, 생년월일, 연락처, 사건번호 등 기관이 요구하는 정보를 비공개 경로로 제출해야 할 수 있습니다.
* 기관별 처리 방식이 다를 수 있으므로 관할 경찰서 또는 면허 담당 부서 안내를 따르십시오.
