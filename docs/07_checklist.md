# 체크리스트

## 접수 전

* [ ] 처분서 또는 응시원서 접수 거부처분서 확보
* [ ] 처분일과 처분을 알게 된 날 확인
* [ ] 행정심판 청구기간 계산
* [ ] 행심24 또는 관할 행정심판위원회 확인
* [ ] 청구취지 작성
* [ ] 청구이유 작성
* [ ] 증거자료 목록 작성
* [ ] 개인정보가 포함된 자료 비공개 보관

## 접수 시

* [ ] 청구인 정보 확인
* [ ] 피청구인 확인
* [ ] 처분명 확인
* [ ] 처분서 첨부
* [ ] 증거자료 첨부
* [ ] 청구취지와 청구이유 최종 확인
* [ ] 접수번호 기록

## 접수 후

* [ ] 피청구인 답변서 송달 여부 확인
* [ ] 답변서 내용 검토
* [ ] 보충의견서 제출 필요성 검토
* [ ] 추가 증거자료 제출
* [ ] 심리 또는 재결 일정 확인

## 재결 후

* [ ] 재결서 다운로드
* [ ] 인용, 기각, 각하 여부 확인
* [ ] 인용 시 경찰서 또는 면허 담당 부서에 재결서 송부
* [ ] 결격 해제 전산 반영 요청
* [ ] 전산 반영 여부 확인
* [ ] 운전면허시험장 방문

* [ ] 전화번호와 이메일 제거
