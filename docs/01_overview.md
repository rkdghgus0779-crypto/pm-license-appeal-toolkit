# 개요

개인형 이동장치(PM, 전동킥보드 등)는 자동차와 동일하지 않지만, 도로교통법상 일정한 운전면허 규율과 안전규율의 대상입니다. 특히 음주운전, 무면허운전, 결격기간, 운전면허시험 응시 제한 문제가 결합되면 일반인이 절차를 이해하기 어렵습니다.

## 왜 문제가 복잡한가

개인형 이동장치는 자동차보다 규모와 위험성이 다를 수 있지만, 도로에서 운행되는 교통수단입니다. 그래서 도로교통법상 운전면허, 음주운전, 무면허운전, 범칙금, 결격기간 문제가 함께 발생할 수 있습니다.

문제가 되는 대표 상황은 다음과 같습니다.

* 개인형 이동장치 음주운전으로 범칙금 통고처분을 받은 뒤 운전면허 결격기간이 문제되는 경우
* 운전면허시험장에서 결격기간 미경과를 이유로 응시원서 접수 거부처분을 받는 경우
* 범칙금 통고처분을 납부한 사람과 즉결심판 절차를 거친 사람 사이의 형평성이 문제되는 경우
* 무면허 상태에서 개인형 이동장치를 운전한 뒤 결격기간이 적용되는 경우

## 행정심판에서 검토할 수 있는 쟁점

행정심판에서 항상 인용되는 것은 아닙니다. 다만 사안에 따라 다음 쟁점이 검토될 수 있습니다.

* 비례원칙: 위반행위의 내용과 처분의 불이익 사이에 균형이 있는지
* 형평성: 유사한 사안과 비교할 때 지나치게 불리한 결과가 발생하는지
* 입법적 흠결 또는 제도 미비: 개인형 이동장치 특성이 충분히 반영되었는지
* 재량권 일탈ㆍ남용 또는 처분의 부당성: 처분이 구체적 사정을 충분히 고려했는지

## 공식 자료 확인

도로교통법상 개인형 이동장치 정의는 국가법령정보센터에서 확인할 수 있습니다. 행정심판 청구기간은 행정심판법 제27조가 기준이 되며, 일반적으로 처분이 있음을 알게 된 날부터 90일, 처분이 있었던 날부터 180일이라는 기준이 중요합니다.
